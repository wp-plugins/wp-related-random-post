Shows the related posts on a single post in same catagory. 

== Description ==

This plugin gives the user ability to show related post on the single post. Related posts are fully dynamic posts and random. User can increase the number of ralated posts too.

= Features:  =

* User can show related related posts on a catagory
* It shows the random posts always




== Installation ==

Very easy to install like other plugins. Just put it on Wp-content>>plugins directory and  go to Dashboard >> Plugins >> activate
